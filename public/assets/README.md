# Assets Folder

Place your profile picture and resume PDF in this folder so that the website can render them and make them downloadable:

1. **Profile Picture:**
   - File name: `profile.jpg`
   - Path: `public/assets/profile.jpg`
   - Size: Recommended size is square (e.g., 400x400 pixels or larger). The UI will style it with rounded-corner corners automatically.

2. **Resume PDF:**
   - File name: `resume.pdf`
   - Path: `public/assets/resume.pdf`
   - Note: Make sure it is exported as a standard PDF file. Clicking the "Download Resume" buttons on the website will trigger a browser download for this file.
